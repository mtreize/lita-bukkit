# lita-bukkit

[Travis-CI Badge]()[Coveralls Badge]()

Grabs a random image from the list of images at [http://bukk.it](http://bukk.it)

## Installation

Add lita-bukkit to your Lita instance's Gemfile:

``` ruby
gem "lita-bukkit"
```

## Configuration

None.

## Usage

```
Lita: bukkit me
```
